    Jobly Frontend (React)
  </h1>
</div>

This is the React frontend for Jobly, a job search application.

### Tech Stack:
* Javascript
* React
* Jest
* React Router

### Instructions:

1. Clone this repository.
2. Run application in development mode:

        npm start
### Testing:
   
* Run all tests (additional tests to be added):

      npm test
