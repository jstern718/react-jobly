This is the React frontend for Jobly, a job search application.

### Tech Stack:
* Javascript
* React
* HTML
* CSS

### Instructions:

1. Clone this repository.
2. Run application in development mode:

        npm start
### Testing:
   
* Run all tests:

      npm test
